ИНТЕРФЕЙС
=========
Драйвер поддерживает LinuxDVB API.
См. /dev/dvb/adapterX/demuxY

Отладочные параметры доступны через debugfs.
См. /sys/bus/debug/mdemux/*

ОШИБКИ
======
Из-за большого количества аппаратных ошибок, поддерживаются только аппаратные
TS-фильтры, работающие на одном ДМА канале. Остальные типы фильтров реализованы
программно.

Одновременно можно использовать только один входной канал (либо
/dev/dvb/adapterX/demux0, либо /dev/dvb/adapterX/demux1).
