ИНТЕРФЕЙС
=========
Пользовательский интерфейс базируется на нестандартном char-device /dev/msci0.
С ним может взаиммодействовать тестовая программа ldecod из проекта msci-test.

ОШИБКИ
======
Обнаружены ошибки в работе, вызванные, скорее всего, проблемами с аппаратурой.

- Прерывание CWT в процессе процедуры PPS (вроде бы не влияющее на дальнейшую
  работу).
- Странная задержка перед передачей 9го байта по Т1.

Выдержка из лога: запись 9-байтной команды по протоколу Т1.
